Reyette Downloader

Reyette DLR adalah aplikasi open-source untuk mengunduh MP3/MP4 dari TikTok, YouTube, Facebook, Instagram, dan X. Dibangun dengan Python & Kivy, serta dapat dijalankan di Android menggunakan Buildozer.

🚀

Cepat

Unduh video/audio dengan performa ringan tanpa framework berat.

📱

Cross Platform

Bisa dijalankan di Windows, Linux, dan Android.

🎨

Modern UI

Antarmuka berbasis Kivy/KivyMD dengan efek glassmorphism.

⚡

Flexible

Mudah dikembangkan menjadi dashboard atau aplikasi web.
